# Votação Tutorando

Um sistema simples para gerenciar votações entre tutorandos.

## Funcionalidades
- Cadastro de usuários
- Criação e gerenciamento de votações
- Registro e apuração de votos

## Requisitos
- NodeJS v24.14.0

## Instalação
```bash
git clone https://github.com/talesmateus123/tutoria_escola.git
cd tutoria_escola
npm install
```

## Uso
```bash
npm start
```
